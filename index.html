Sobre Mí

¡Hola! Soy Dario, un desarrollador web apasionado por crear cosas que viven en internet. Mi interés en el desarrollo web comenzó cuando decidí crear mi primera página web — resultó que personalizar un tema de WordPress me enseñó mucho sobre HTML y CSS.

Hoy en día, he tenido el privilegio de trabajar en diversos proyectos, desde sitios web corporativos hasta aplicaciones web complejas. Mi enfoque principal es construir productos digitales accesibles e inclusivos para una variedad de clientes.

Estas son algunas tecnologías con las que he estado trabajando recientemente:

  • JavaScript (ES6+)
  • TypeScript
  • React
  • Next.js
  • Node.js
  • Tailwind CSS
  • PostgreSQL
  • Git
